# 🎮 Desenvolvedora e Bugs

Projeto desenvolvido para a disciplina de Desenvolvimento de Jogos do curso de Análise e Desenvolvimento de Sistemas.

## 📖 Sobre o jogo

Neste jogo, o objetivo é sobreviver aos bugs e conquistar a maior pontuação possível.

Este projeto foi desenvolvido para colocar em prática conceitos de programação utilizando Python e Pygame.

## 🛠 Tecnologias utilizadas

- Python
- Pygame

## ▶️ Como executar

1. Clone este repositório:

```bash
git clone https://github.com/fedossijk/jogo-faculdade-python.git
```

2. Instale as dependências:

```bash
pip install pygame
```

3. Execute o jogo:

```bash
python main.py
```
